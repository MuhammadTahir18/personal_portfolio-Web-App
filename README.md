# Personal Portfolio App (Flutter)

A cross-platform personal portfolio application built using Flutter.
The app works on Android, iOS, and Web with a clean architecture and responsive UI.

## Features
- Cross-platform support (Android, iOS, Web)
- MVVM architecture
- Firebase Firestore integration
- Provider state management
- Responsive design for all screen sizes
- Clean and maintainable codebase

## Tech Stack
- Flutter (Dart)
- MVVM architecture
- Firebase Firestore
- Provider
- Material UI

## Project Structure
lib/
- models
- views
- viewmodels
- services
- utils
- main.dart

## Architecture
The project follows the MVVM (Model-View-ViewModel) pattern.
This ensures separation of concerns, better scalability, and easier maintenance.

## Firebase
- Firestore used for storing portfolio data
- Real-time updates
- Scalable cloud backend

## Responsiveness
The UI adapts to different screen sizes including mobile, tablet, and web.
Responsive widgets and flexible layouts are used throughout the app.

## Platforms Tested
- Android
- iOS
- Web

## Purpose
This project was built to demonstrate professional Flutter development,
clean architecture, and real-world app structure for portfolio and interviews.
